# Drawgotchi

Drawgotchi his a virtual pet game where players care for a creature by drawing food and bed.
A CNN model recognizes doodles in real time, while Unity handles gameplay, creature states, and UI.

## Overview

This project was created for the University of Twente Creative Technology Module 7: Hands-on AI.
It combines a Python-based doodle classifier with a Unity game inspired by Tamagotchi and Nintendo DS interfaces.

## Strucrue

- 'python/': CNN model, preprocessing, and HTTP server
- 'unity/': game logic, UI, creatures states, and interactions
